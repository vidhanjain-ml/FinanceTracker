# Finance-Tracker-AI-Integrated
A AI integrated finance tracker with rich colors,that keeps you ahead of your expenses.
A sleek, color-coded command-line interface (CLI) application for tracking personal finances, complete with smart AI-driven budgeting insights.

Features
* Expense Management: Easily add, view, search, and delete transactions.
* Structured Storage: Saves all financial data safely to a local JSON file.
* Beautiful UI: Styled using the rich Python library for clear, color-coded terminal interactions.
* AI Insights: Uses AI to analyze your spending habits and provide personalized tips.
